# About

Python wrapper for [Google Translate Speech Synthesis](https://gtts.readthedocs.io/en/latest/index.html).

## Supported Platforms
- Ubuntu
- Windows

## Installation

### Ubuntu
```
$ apt-get install ffmpeg libavcodec-extra
$ pip install gTTS==2.0.3 pydub
```
